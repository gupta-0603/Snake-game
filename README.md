# 🐍 Snake Game

A classic Snake Game built using **Python** and **Pygame**. The player controls a snake that grows in length as it eats food while avoiding collisions with itself and the game boundaries.


## 🚀 Features

- Snake movement using keyboard controls
- Random food generation
- Real-time score tracking
- Collision detection
- Game Over screen
- Smooth gameplay using Pygame

## 🛠️ Technologies Used

- Python 3
- Pygame

## 📂 Project Structure

```
Snake-game/

```

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/gupta-0603/Snake-game.git
```

2. Navigate to the project directory:

```bash
cd Snake-game
```

3. Install the required library:

```bash
pip install pygame
```

## ▶️ Running the Game

Run the following command:

```bash
python snake1.py
```

## 🎮 Controls

| Key | Action |
|------|---------|
| ↑ | Move Up |
| ↓ | Move Down |
| ← | Move Left |
| → | Move Right |
